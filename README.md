# rock-paper-scissors-java
- A simple command-line implementation of the classic Rock Paper Scissors game built in Java. The user competes against the computer, whose moves are generated randomly.
  
## Features
- Interactive console gameplay
- Random computer moves
- Win, lose, and tie detection
  
## Technologies
- Java
- Scanner
- Math.random()
- if-else & switch statements
